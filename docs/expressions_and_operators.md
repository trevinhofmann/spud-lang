# Expressions & Operators

## Assignment

* `=`

### Shorthand

* `x += y` => `x = x + y`
* `x *= y` => `x = x *+* y`
* `x -= y` => `x = x - y`
* `x /= y` => `x = x / y`
* `x **= y` => `x = x ** y`

## Arithmetic

* `x + y` (addition)
* `x * y` (muliplication)
* `x - y` (subtraction)
* `x / y` (division)
* `x ** y` (exponentiation)
* `+x` (identity, does not change the value of `x`)
* `-x` (negative)

## Comparison

* `x < y` (less than)
* `x <= y` (less than or equal to)
* `x >= y` (greater than or equal to)
* `x > y` (greater than)

* `x == y` (equal to)
* `x != y` (not equal to)

## Logical Operators

* `!x` (not)
* `x && y` (and)
* `x || y` (or)

## Grouping

* `(x + y)` (evaluate inside parentheses first)
