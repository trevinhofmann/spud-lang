# Control Flow

## Conditions

* `if`
* `else if`
* `else`

```spud
if condition
  // statement
  // statement
else if condition
  // statement
  // statement
else
  // statement
  // statement
```

## Loops

### For-loops

```spud
uint x = 0

for i in {0 -> 100}
  x += i
```

### While-loops

```spud
while condition
  // statement
  // statement
```
